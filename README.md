## 🎯 Quick Start

1. Clone the repository
2. Copy `config/.env.example` to `.env` and add your API keys
3. Follow the [Google Maps Setup Guide](docs/setup/GOOGLE_MAPS_SETUP.md)
4. Open `indexMVP.html` in your browser
